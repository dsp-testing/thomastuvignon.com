Génère un shortcode avec les mentions légales obligatoires sur les sites internet français.

== Description ==

Ce plugin permet à l'administrateur d'un site WordPress d'afficher les mentions légales obligatoires, par l'utilisation d'un shortcode.
L'utilisateur a juste à inscrire les informations du site, et de copier le shortcode sur une page dénommée par exemple "Mentions légales".

= Fonctionnalités =
 * Génération des mentions légales pour les sites professionnels
 * Génération des mentions légales pour les sites particuliers
 * Création d'un widget permettant l'affichage d'un lien vers votre page des mentions légales

= Usage =
 1. Aller dans Réglages -> Mentions légales
 2. Remplir le formulaire avec les informations adéquates
 3. Copier le shortcode qui apparaît, et coller dans une nouvelle page WordPress dénommée "Mentions légales"
 4. Utilisez éventuellement le widget *Mentions légales* pour afficher un lien vers cette page sur tout votre site

== Installation ==

= Depuis le tableau de bord =
1. Aller dans *Plugins* et cliquer sur *Ajouter*
2. Chercher *Mentions légales*
3. Cliquer sur *Installer* sur la fiche du plugin
4. Activer le plugin
 
== Frequently Asked Questions ==

= Quelles types de mentions légales génère ce plugin ? =

Cette extension génère les mentions légales adaptées aux sites professionnels ou particuliers. Il est adapté à la majorité des cas, néanmoins des mentions spécifiques peuvent être demandées selon votre activité. Dans ce cas, malheureusement pour le moment, il est impossible de les ajouter.

= Je suis auto-entrepreneur ou micro-entrepreneur, et je ne trouve pas le choix dans la liste déroulante =

Les auto-entrepreneurs et micro-entrepreneurs font partie des Entreprises Individuelles (EI). Leur statut leur permet simplement une gestion simplifiée. Choisissez *Entreprise Individuelle (EI)*.

= Je ne trouve pas le shortcode à insérer dans ma page "mentions légales" =

Pour afficher le shortcode à insérer dans votre page, il faut saisir toutes les informations nécessaires à la génération des mentions. Vérifiez que toutes les informations sont remplies, et enregistrez. Le shortcode doit alors apparaitre sur la même page.

= Je ne trouve pas le widget dans la liste de ceux disponibles =

Pour que vous puissiez utiliser le widget qui permet d'afficher un lien vers la page des mentions légales, il faut que vous ayez sélectionné dans la liste déroulante sur quelle page vous avez inséré le shortcode. Sans cela, il est impossible de générer le lien, et donc le widget est inutilisable.
